Question #4

Across the United States, how have emissions from coal combustion-related sources changed from 1999-2008?

The strategy Ill use to extract the emissions of coal related sources is ad follows:

  • Using a regular expression (.*[cC]oal.*) well match the values of SCCs EI.Sector columns that are coal-related. The reg exp means:
  • .*" matches any number () of any characters (.). Read it asany character (.) as many times () as youd like
  • [cC] matches either the lowercase c or uppercase C just once
  • oal matches exactly that series of characters

  • .*" matches any character.

  • the corresponding values of SCC are extracted;

  • these values are then compared with NEIs SCC columns and matching rows are selected.

Lets see this in action:

coal_related_EI.Sectors <- unique(grep('.*[cC]oal.*', SCC$EI.Sector, ignore.case = TRUE, value = TRUE))
coal_related <- SCC$EI.Sector %in% coal_related_EI.Sectors
length(coal_related)

## [1] TRUE TRUE TRUE TRUE TRUE TRUE

Only 99 rows of SCC dataset (0.0084 %) are coal-related. Now get the corresponding SCC codes, and use them to subset the NEI dataset:
